The Code for Taco Cat.

                            
//Retrieve value from user
//Remove special characters and white space
function getValue() {
    let inputValue = document.getElementById('userString').value;
    let parsed = inputValue.normalize('NFC').replace(/[^0-9a-zA-Z]/g, '').toLowerCase();
    let {palindrome, reversed} = isPalindrome(parsed);
    displayResults(palindrome, reversed);
}
function isPalindrome(parsed) {
    //build array in reverse and check against forward
    let reversed = '';
    let palindrome = false;
    for (let i = parsed.length - 1; i >= 0; i--) {
        reversed += parsed[i];
    }
    if(reversed == parsed){palindrome = true;}
    return {palindrome, reversed};
}
//Display results to user
function displayResults(palindrome, reversed) {
    if(palindrome) {document.getElementById('alertHeading').innerHTML = 'Well done! You entered a palindrome!';}
    else {document.getElementById('alertHeading').innerHTML = 'Im sorry. Your value is not a palindrome.';}
    document.getElementById("msg").innerHTML = `Your phrase reversed is: ${reversed}`;
    document.getElementById("alert").classList.remove('invisible');
}

Taco Cat is structured in 3 main functions:

getValue()

The first function retrieves the value entered by the user and uses a regex to remove any special characters and white space. It then sets all characters to lowercase before sending the value to the second function, isPalindrome...

isPalindrome()

The second function declares two variables initially. One for our parsed value once it has been reversed and one that is a boolean that we will use to tell the last function whether we have a palindrome or not. Our function then uses a For Loop to reverse our value and set the reversed value to our variable. Then it sets our boolean variable to the appropriate value based on whether or not we have a palindrome. Finally, it returns both the reversed value and the boolean so they can be used in the final function...

displayResults()

Our last function receives the reversed value and the boolean and first sets the HTML of our alertHeading to the appropriate text based on the value of the boolean and then sets the HTML of our msg with the reversed value. Lastly, it makes our alert section visible by removing the invisible class attribute.
